# Firemní diagnostika — AB ANTE

Landing page pro službu provozní a ekonomické diagnostiky firem AB ANTE s.r.o.

**Live:** [firemni-diagnostika.vercel.app](https://firemni-diagnostika.vercel.app)

## Stack

Static HTML/CSS/JS — bez frameworku, bez build stepu. Auto-deploy z `main` na Vercel.

## Design

Sdílený design systém s [krizova-diagnostika.vercel.app](https://krizova-diagnostika.vercel.app):

- **Fonty:** Source Serif 4 (nadpisy) + DM Sans (body)
- **Barvy:** Navy `#1A2E4A` · Copper `#D4722A` · Cream `#FAFAF8`
- **Responsive:** Desktop / Tablet (960px) / Mobile (640px)
